# Stardust System Heartbeat - Galactic Note Ring Overview

## Overview
This directory contains variants of the Galactic Note Ring Overview interface, a visualization tool for managing interconnected notes in a cosmic-themed interface.

## Structure
- `common_components/` - Shared resources between all variants
  - `common_functions.js` - JavaScript utility functions
  - `common_styles.css` - Common CSS styles
- `galactic_note_ring_overview_variant_1_notes_choice/` - Notes selection interface
- `galactic_note_ring_overview_variant_2_linking_notes/` - Linking/connections interface

## Variants

### Variant 1: Notes Choice
A planetary ring interface focused on individual note selection and organization. Features:
- Planetary system visualization
- Individual note selection with hover effects
- Dark/light mode toggle
- Navigation controls

### Variant 2: Linking Notes
A research web interface focused on connecting notes and visualizing relationships. Features:
- Interconnected node visualization
- Connection pathways between concepts
- Pulsar and planet nodes representing different types of information
- Glass-morphism design elements

## Common Components
Both variants share:
- Theme management (dark/light mode)
- Navigation controls
- Responsive design principles
- Material Icons integration

## Usage
Each variant can be opened directly by opening the `code.html` file in a browser.

To modify:
1. Common functionality: Edit files in `common_components/`
2. Variant-specific features: Edit the respective variant folder's `code.html` file
